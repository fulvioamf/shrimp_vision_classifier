# crab_vision_classifier

Este projeto possui uma parcela dos dados utilizados durante a pesquisa realizada pelo
Laboratório de Ecologia e Evolução de Crustáceos (LABEEC) da UFRN. Sendo composto
por uma série de imagens de camarões do gênero Xiphopenaeus em escalas de cinza para o treinamento
de redes neurais para a classificação de diferentes espécies, visando observar sua forma de aprendizado
a partir das características anatômicas dos animais.
